---
title: 'Create Company with InvestorMatch'
openapi: 'POST /api/v1/company/create-with-investormatch'
description: 'Register a UK company with comprehensive investor matching data'
---

<Note>
  This endpoint requires authentication via X-API-Key header. Contact support@diligentsia.com to obtain your partner API key.
</Note>

<Info>
  **New Feature**: The `sendEmail` flag (optional, default: `true`) controls email notifications:
  - When `true`: User receives an activation email with login instructions
  - When `false`: API returns a `loginLink` in the response for direct access
</Info>

<RequestExample>
```json Request with Email
{
  "sendEmail": true,
  "user": {
    "email": "john.doe@example.com",
    "firstName": "John",
    "lastName": "Doe"
  },
  "company": {
    "ukRegNo": "12345678",
    "companyWebsite": "https://techstartup.co.uk",
    "company5Words": "Innovative software development consulting services"
  },
  "investorMatch": {
    "sector": "Software",
    "stage": "Post-Revenue Early Stage",
    "annualRevenues": "£1m-£5m",
    "annualProfits": "£100'000-£500'000",
    "region": "Greater London",
    "sweetSpot": "£1'000'001-£5'000'000"
  }
}
```
