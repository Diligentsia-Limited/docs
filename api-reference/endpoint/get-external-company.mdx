---
title: 'Get External Company'
openapi: 'GET /api/v1/company/{ukRegNo}'
description: 'Retrieve company information with comprehensive investor matching data'
---

<Note>
  This endpoint requires authentication via Bearer JWT token. Include your JWT token in the Authorization header.
</Note>

<Info>
  This endpoint returns comprehensive company data including investor matching scores and detailed fit reports when InvestorMatch data is available.
</Info>

<RequestExample>
```bash cURL
curl -X GET \
  https://devserver.diligentsia.co.uk/api/v1/company/90654233 \
  -H 'Authorization: Bearer YOUR_JWT_TOKEN'
```

```javascript JavaScript
const response = await fetch('https://devserver.diligentsia.co.uk/api/v1/company/90654233', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
}

response = requests.get(
    'https://devserver.diligentsia.co.uk/api/v1/company/90654233',
    headers=headers
)

data = response.json()
print(data)
```
</RequestExample>
