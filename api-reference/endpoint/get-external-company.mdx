---
title: 'Get Partner Company'
openapi: 'GET /api/v1/company/{ukRegNo}'
description: 'Retrieve company information with comprehensive investor matching data'
---

<Note>
  This endpoint requires authentication via X-API-Key header. Contact support@diligentsia.co.uk to obtain your partner API key.
</Note>

<Info>
  This endpoint returns comprehensive company data including investor matching scores and detailed fit reports when InvestorMatch data is available.
</Info>

<RequestExample>
```bash cURL
curl -X GET \
  https://devserver.diligentsia.co.uk/api/v1/company/90654233 \
  -H 'X-API-Key: YOUR_API_KEY'
```

```javascript JavaScript
const response = await fetch('https://devserver.diligentsia.co.uk/api/v1/company/90654233', {
  method: 'GET',
  headers: {
    'X-API-Key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

headers = {
    'X-API-Key': 'YOUR_API_KEY'
}

response = requests.get(
    'https://devserver.diligentsia.co.uk/api/v1/company/90654233',
    headers=headers
)

data = response.json()
print(data)
```
</RequestExample>
