---
title: 'Get Partner Company'
openapi: 'GET /api/v1/company'
description: 'Retrieve company information with comprehensive investor matching data'
---

<Note>
  This endpoint requires authentication via X-API-Key header. Contact support@diligentsia.co.uk to obtain your partner API key.
</Note>

<Info>
  This endpoint returns comprehensive company data including investor matching scores, detailed fit reports, and KPI reports when InvestorMatch data is available. The endpoint supports query parameter format with optional trailing slash: `/api/v1/company?ukRegNo=03237591` or `/api/v1/company/?ukRegNo=03237591`
</Info>

<RequestExample>
```bash cURL
curl -X GET \
  'https://prodserver.diligentsia.co.uk/api/v1/company?ukRegNo=03237591' \
  -H 'X-API-Key: YOUR_API_KEY'
```

```javascript JavaScript
const response = await fetch('https://prodserver.diligentsia.co.uk/api/v1/company?ukRegNo=03237591', {
  method: 'GET',
  headers: {
    'X-API-Key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

headers = {
    'X-API-Key': 'YOUR_API_KEY'
}

response = requests.get(
    'https://prodserver.diligentsia.co.uk/api/v1/company',
    params={'ukRegNo': '03237591'},
    headers=headers
)

data = response.json()
print(data)
```
</RequestExample>
