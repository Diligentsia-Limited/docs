---
title: 'Upload Pitch Deck'
openapi: 'POST /api/v1/company/upload-pitch-deck'
description: 'Upload pitch deck files for a registered company'
---

<Note>
  This endpoint requires authentication via X-API-Key header. Contact support@diligentsia.co.uk to obtain your partner API key.
</Note>

<Warning>
  This endpoint uses `multipart/form-data` encoding, not JSON. Files must be uploaded as form data.
</Warning>

<Info>
  **Request Headers:**
  - `X-API-Key` (required) - Your API authentication key
  - `Content-Type` (required) - Must be `multipart/form-data`

  **Request Body (form-data):**
  - `companyId` (string, required) - The MongoDB ObjectId of the registered company
  - `files` (file(s), required) - One or more pitch deck files to upload

  **Supported File Types:**
  - PDF (recommended)
  - PowerPoint (.pptx, .ppt)
  - Images (PNG, JPG, JPEG)
  - Documents (DOCX, DOC)
</Info>

<Tip>
  The `companyId` can be obtained from the response of the Company Registration API (`/api/v1/company/create-with-investormatch`).
</Tip>

<RequestExample>
```bash cURL (Single File)
curl -X POST "https://prodserver.diligentsia.co.uk/api/v1/company/upload-pitch-deck" \
  -H "X-API-Key: your-api-key-here" \
  -H "Content-Type: multipart/form-data" \
  -F "companyId=60a1b2c3d4e5f6a7b8c9d0e1" \
  -F "files=@/path/to/pitch-deck.pdf"
```


```javascript JavaScript (FormData)
const formData = new FormData();
formData.append('companyId', '60a1b2c3d4e5f6a7b8c9d0e1');
```

```python Python (requests)
import requests

url = "https://prodserver.diligentsia.co.uk/api/v1/company/upload-pitch-deck"
headers = {
    "X-API-Key": "your-api-key-here"
}
files = {
    "files": open("/path/to/pitch-deck.pdf", "rb")
}
data = {
    "companyId": "60a1b2c3d4e5f6a7b8c9d0e1"
}

response = requests.post(url, headers=headers, files=files, data=data)
print(response.json())
```
