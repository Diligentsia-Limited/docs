## 🔐 Authentication

The create company with investor match endpoint requires authentication using an API key provided in the request headers.

### API Key Header

Every request to the API must include your Partner API key in the headers:

<CodeGroup>

```bash Header Format
X-API-Key: your-partner-api-key-here
```

</CodeGroup>

<Warning>
  Never expose your API key in client-side code or public repositories. Always keep it secure on your server.
</Warning>

## 📋 Required Headers

<Tabs>
  <Tab title="Headers Table">
    
    | Header | Type | Description |
    |--------|------|-------------|
    | `X-API-Key` | string | Your partner API key for authentication |
    | `Content-Type` | string | Should be `application/json` for POST requests |
    
  </Tab>
  <Tab title="Example Headers">
  
    ```json
    {
      "X-API-Key": "your-partner-api-key",
      "Content-Type": "application/json"
    }
    ```
    
  </Tab>
</Tabs>
