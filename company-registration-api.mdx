## 🚀 Overview

The Company Registration API enables partners to seamlessly register UK companies and automatically match them with potential investors. This powerful endpoint integrates with Companies House data and provides instant investor compatibility scoring based on multiple business metrics.

## 📍 Endpoint Details

<Card title="POST /api/v1/company/create-with-investormatch" icon="paper-plane">
  
  **Method:** `POST`
  
  **URL:** `https://devserver.diligentsia.co.uk/api/v1/company/create-with-investormatch`
  
  **Authentication:** Required (X-API-Key header)
  
  **Content-Type:** `application/json`
  
</Card>

## 📝 Request Body Structure

The request body consists of three main sections:

<Tabs>
  <Tab title="User Information">
    
    <Note>
      **Optional Section** - If not provided, a guest user will be created
    </Note>
    
    ### User Fields
    
    | Field | Type | Required | Description | Default |
    |-------|------|----------|-------------|----------|
    | `sendEmail` | boolean | No | Controls whether to send activation email | `true` |
    | `email` | string | No | User's email address for account creation | - |
    | `firstName` | string | No | User's first name | - |
    | `lastName` | string | No | User's last name | - |
    
    <Info>
      **Email Behavior:**
      - If `sendEmail` is `true` (default): User receives an activation email with login instructions
      - If `sendEmail` is `false`: API returns a `loginLink` in the response for direct access
    </Info>
    
  </Tab>
  
  <Tab title="Company Details">
    
    <Warning>
      **Required Section** - All company fields must be provided
    </Warning>
    
    ### Company Fields
    
    | Field | Type | Required | Description | Validation |
    |-------|------|----------|-------------|------------|
    | `ukRegNo` | string | **Yes** | UK company registration number | 6-8 alphanumeric characters |
    | `companyWebsite` | string | **Yes** | Company website URL | Valid URL format |
    | `company5Words` | string | **Yes** | Company description | Minimum 5 words |
    
    <Tip>
      UK registration numbers are automatically normalized (spaces removed, uppercase)
    </Tip>
    
  </Tab>
  
  <Tab title="InvestorMatch Data">
    
    <Warning>
      **Required Section** - All InvestorMatch fields must be provided
    </Warning>
    
    ### InvestorMatch Fields
    
    | Field | Type | Required | Description |
    |-------|------|----------|-------------|
    | `sector` | enum | **Yes** | Business sector |
    | `stage` | enum | **Yes** | Company development stage |
    | `annualRevenues` | enum | **Yes** | Current year revenue range |
    | `annualProfits` | enum | **Yes** | Current year profit range |
    | `region` | enum | **Yes** | UK operating region |
    | `sweetSpot` | enum | **Yes** | Investment amount sought |
    
  </Tab>
</Tabs>

## ✅ Validation Rules

### 🏢 Company Validation

<AccordionGroup>
  <Accordion title="UK Registration Number">
    - **Format:** 6-8 alphanumeric characters
    - **Pattern:** `/^[0-9A-Z]{6,8}$/`
    - **Auto-transformation:** Spaces removed, converted to uppercase
    - **Examples:** `12345678`, `AB123456`, `SC123456`
  </Accordion>
  
  <Accordion title="Company Website">
    - **Format:** Valid URL
    - **Protocol:** Optional (http/https added if missing)
    - **Examples:** `example.com`, `https://example.co.uk`
  </Accordion>
  
  <Accordion title="Company Description">
    - **Minimum:** 5 words
    - **Separator:** Space-separated words
    - **Example:** `Software development consulting services company`
  </Accordion>
</AccordionGroup>

## 🎯 What Happens Next?

<Steps>
  <Step title="Company Creation">
    Company profile is created with Companies House integration
  </Step>
  <Step title="User Account">
    User account is created or updated with partner association
  </Step>
  <Step title="Email Notification or Login Link">
    - If `sendEmail` is `true`: User receives activation email with permission options
    - If `sendEmail` is `false`: API returns `loginLink` for direct access
  </Step>
  <Step title="Investor Matching">
    System matches company against 1000+ investor profiles
  </Step>
  <Step title="Fit Score Calculation">
    Compatibility scores calculated based on 6 key metrics
  </Step>
  <Step title="Report Generation">
    Top 10 investor matches returned with detailed scoring
  </Step>
</Steps>
